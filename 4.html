Web Hosting

  • github라는 무료로 호스트 해주는 업체가 있다.
    index파일을 만들어줘야 메인페이지라는걸 인식시켜줄 수 있다.
    미리 메인페이지가 될 파일명을 index.html로 바꾸자.

    1. -회원가입을한다.
    2. -프로필 아이콘 옆에 +버튼을 누른뒤 New repository 선택
    3. -Repository name에 html파일들이 모여있는 상위폴더이름을 적어준다.
      지금파일이름은 web인데 에러가뜰 가능성이 있으니 .html라고 뒤에 붙여주자 ex)web.html
    4. -Initialize this repository with a rename을 체크 (Add a readme file체크)
    5. -create repository클릭
    6. -Upload files->Crate file
    7. -작업한 파일들 드래그드롭
    8. -Add files via upload칸에 어떤내용을 담고있다 설명칸 ex)first version
      (그냥 적당히 써줘도 됨)
    9. -Commit changes클릭
      수정할때-Upload files해서 파일 똑같이 드래그드롭해서 올리고
      수정한내용 설명칸에 쓰고 commit changes클릭
    10. -상단에 settings들어가서 내리면 GitHub Pages의 Source밑에 none으로 되있는걸 main으로 바꾸기
      (원래는 master branch였는데 main branch로 이름 바꾸기)
    11. -Your site is published at 주소 라고 뜨는데 주소를 클릭하면 내가만든 웹사이트로 이동된다!
